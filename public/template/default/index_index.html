  • 公司介绍

    上海雷洛资产管理有限公司是一家私募股权、新三板投资类公司，是中国证券投资基 金业协会会员，并获中国证券业协会批准成为 IPO 询价对象。与国内多家大型金融机构合作发 行并管理多支证券类信托基金，投研团队为券商自营、投行、私募基金和市场派的完美组合，从 容应对市场复杂变化！公司投资方向为主动管理、量化对冲、程序化交易等高端财富管理领域， 在行业内享有盛誉！雷洛资产专注于中国股票市场，长期业绩优异！
